# Cricket Score Broadcast Suite

A minimal, colorful web application for managing, scoring, and broadcasting live cricket matches.

**[Live Preview](https://expensestracker.github.io/live/)**

## Modules

*   **Scoring** (`score.html`): For entering scores.
*   **View & Stats** (`view.html`): View scores and statistics.
*   **OBS Output** (`live.html`): Transparent live feed for OBS/vMix.
*   **Broadcast Control** (`broadcast.html`): Control scenes and custom graphics.
